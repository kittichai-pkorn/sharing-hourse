# ประเภทวงแชร์ (Group Types)

## ภาพรวม

ระบบรองรับวงแชร์ 3 ประเภทหลัก:

| ประเภท | ชื่อไทย | คำอธิบาย |
|--------|---------|----------|
| STEP_INTEREST | ขั้นบันได | ลูกแชร์แต่ละคนมียอดส่งคงที่ต่างกัน |
| FIXED_INTEREST | ดอกตาม | ลูกแชร์ส่งเท่ากัน + ดอกคงที่หลังเปีย |
| BID_INTEREST | บิทดอกตาม | ลูกแชร์ประมูลดอก คนให้สูงสุดชนะ |

---

## การกำหนดผู้เปีย

| หัวข้อ | STEP_INTEREST | FIXED_INTEREST | BID_INTEREST |
|--------|---------------|----------------|--------------|
| **วิธีกำหนดผู้เปีย** | ท้าวเลือก (Dropdown) | ท้าวเลือก (Dropdown) | ท้าวเลือก (Dropdown) |
| **เลือกเมื่อไหร่** | ก่อนเปิดวง | ก่อนเปิดวง | ก่อนเปิดวง |
| **เงื่อนไขเปิดวง** | ต้องเลือกครบทุกงวด | ต้องเลือกครบทุกงวด | ต้องเลือกครบทุกงวด |
| **งวดแรก** | ท้าว (อัตโนมัติ) | ท้าว (อัตโนมัติ) | ท้าว (อัตโนมัติ) |
| **งวดสุดท้าย** | ท้าว (หักท้าย) | คนที่เหลือ | คนที่เหลือ |

---

## ยอดส่งหลังเปีย

| ประเภท | ก่อนเปีย | หลังเปีย |
|--------|----------|----------|
| STEP_INTEREST | ยอดคงที่ (ต่างกันแต่ละคน) | ยอดคงที่ (เหมือนเดิม) |
| FIXED_INTEREST | ส่งต่องวด | ส่งต่องวด + ดอกคงที่ |
| BID_INTEREST | ส่งต่องวด | ส่งต่องวด + ดอกเปีย |

---

## รายได้ท้าว

| ประเภท | แหล่งรายได้ |
|--------|-------------|
| STEP_INTEREST | งวดแรก + งวดหักท้าย |
| FIXED_INTEREST | งวดแรก + ดอกเบี้ยทุกงวด + ค่าดูแลวง |
| BID_INTEREST | งวดแรก + ดอกประมูลทุกงวด + ค่าดูแลวง |
