# STEP_INTEREST (ขั้นบันได)

## ลักษณะเฉพาะ

ลูกแชร์แต่ละคนมี **ยอดส่งคงที่ต่างกัน** ส่งทุกงวดจนจบวง

---

## ฟิลด์ตั้งค่าวง

- **ชื่อวงแชร์** * - ชื่อวง
- **รูปแบบวงแชร์** * - เลือก "ขั้นบันได"
- **จำนวนเงินต้น** * - เงินต้นต่องวด (บาท)
- **จำนวนมือ** * - จำนวนสมาชิก
- **รอบชำระ** * - รายวัน/รายสัปดาห์/รายเดือน
- **ระยะห่างงวด** - จำนวนวัน (สำหรับรอบชำระรายวัน)
- **วันที่เริ่มต้น** * - วันเริ่มงวดแรก
- **ค่าดูแลวง** - ค่าบริหารจัดการ (บาท)
- **หักคำท้าย** - จำนวนงวดท้ายที่จะหัก (ค่าตอบแทนท้าว)
- **หมายเหตุ** - บันทึกเพิ่มเติม

> (*) = บังคับกรอก

---

## ฟิลด์รายสมาชิก

- **ยอดส่ง** * - ยอดที่ลูกแชร์ส่งทุกงวด (คงที่ตลอดวง)

> ท้าวแชร์ไม่มียอดส่ง (= 0)

---

## กฎ

1. ท้าวไม่ต้องจ่าย รับเงินอย่างเดียว
2. ลูกแชร์จ่ายยอดส่งเท่าเดิมทุกงวด
3. แต่ละคนมียอดส่งไม่เท่ากัน (ขั้นบันได)
4. ท้าวได้งวดแรก

---

## สูตร

ยอดรับลูกแชร์ = เงินต้น - ยอดส่งคนอื่น - ยอดหักท้าย

ยอดรับท้าว = ผลรวมยอดส่งลูกแชร์ทั้งหมด

---

## ตัวอย่าง

วง: เงินต้น 1,000 บาท, 5 มือ

สมาชิก:
- ป้าแดง (ท้าว) - ยอดส่ง 0
- เต้ - ยอดส่ง 240/งวด
- เต้ง - ยอดส่ง 220/งวด
- ทิม - ยอดส่ง 180/งวด

การคำนวณ:
- งวด 1: ท้าวรับ = 240+220+180 = **640 บาท**
- งวด 2: เต้รับ = 1000-(220+180)-240 = **360 บาท**
- งวด 3: เต้งรับ = 1000-(240+180)-220 = **360 บาท**
- งวด 4: ทิมรับ = 1000-(240+220)-180 = **360 บาท**
