# FIXED_INTEREST (ดอกตาม)

## ลักษณะเฉพาะ

ลูกแชร์ทุกคนส่ง **ยอดเท่ากัน** ทุกงวด และมี **ดอกเบี้ยคงที่** (ยกเว้นงวดที่ตัวเองเปีย)

---

## ฟิลด์ตั้งค่าวง

- **ชื่อวงแชร์** * - ชื่อวง
- **รูปแบบวงแชร์** * - เลือก "ดอกตาม"
- **ดอกเบี้ย** * - ดอกเบี้ยคงที่ต่องวด (บาท)
- **ส่งต่องวด** * - ยอดที่ทุกคนส่งทุกงวด (บาท)
- **จำนวนเงินต้น** * - เงินต้นต่องวด (บาท)
- **จำนวนมือ** * - จำนวนสมาชิก
- **รอบชำระ** * - รายวัน/รายสัปดาห์/รายเดือน
- **ระยะห่างงวด** - จำนวนวัน (สำหรับรอบชำระรายวัน)
- **วันที่เริ่มต้น** * - วันเริ่มงวดแรก
- **ค่าดูแลวง** - ค่าบริหารจัดการ (บาท)
- **หมายเหตุ** - บันทึกเพิ่มเติม

> (*) = บังคับกรอก

---

## กฎ

### ผู้เปีย
| งวด | ใครได้ | วิธี |
|-----|--------|------|
| งวดแรก | ท้าว | ได้อัตโนมัติ |
| งวด 2+ | ลูกแชร์ | ท้าวเลือก (ไม่มีประมูล) |
| งวดสุดท้าย | คนที่เหลือ | ได้อัตโนมัติ |

### ยอดส่ง
| สถานะ | ยอดส่ง |
|-------|--------|
| ท้าว | ไม่ต้องส่งทุกงวด |
| ลูกแชร์ยังไม่เปีย | ส่งต่องวด |
| ลูกแชร์เปียแล้ว | ส่งต่องวด + ดอกเบี้ย |

> หมายเหตุ: ผู้เปียงวดนั้นไม่ต้องส่ง
