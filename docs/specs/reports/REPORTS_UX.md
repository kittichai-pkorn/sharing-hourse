# Reports UX Design (รายงาน)

## Screen Flow

```
Dashboard (รายงาน)
├── Summary Card
├── Quick Links (รายวง / รายคน)
└── Overdue List
    │
    ├──→ รายงานรายวง
    │    ├── Search + Filter
    │    └── Group Cards
    │        └──→ Group Detail
    │            ├── Tab รายงวด
    │            └── Tab รายคน
    │
    └──→ รายงานรายคน
         ├── Search + Filter
         └── Member Cards
             └──→ Member Detail
                 ├── Overdue Section
                 ├── Groups Section
                 └── Payment History
```
