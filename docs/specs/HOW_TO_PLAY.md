# Core Business - วงแชร์

---

## วงแชร์คืออะไร?

**ออมเงินแบบกลุ่ม** - สมาชิกส่งเงินทุกงวด ผลัดกันรับเงินก้อน

---

## คำศัพท์

| คำ | ความหมาย |
|----|----------|
| **ท้าวแชร์** | หัวหน้าวง จัดการเก็บเงิน |
| **ลูกแชร์** | สมาชิก |
| **งวด/มือ** | รอบการจ่ายเงิน |
| **เปีย** | ได้รับเงินก้อน |
| **ดอกเปีย** | เงินที่เสนอประมูล |
| **หักท้าย** | ยอดส่งของสูกแชร์งวดสุดท้ายที่จะถูกหักตอนงวดที่จะรับเงิน 

---

## กฎหลัก

| กฎ | รายละเอียด |
|----|------------|
| จำนวนงวด = จำนวนสมาชิก | 5 คน = 5 งวด |
| จำนวนงวด payment ของลูกแชร์ = จำนวนสมาชิก | 5 คน = 5 งวด |
| งวดแรก | ท้าวรับเงินจากงวดแรกของลูกแชร์ทั้งหมด |
| งวดถัดไป | Admin บันทึกผล |
| งวดสุดท้าย | จะได้รับเงินทั้งหมดจากที่ลูกแชร์ที่จ่าย|

---

## การเลือกผู้ชนะ

**ประมูล/จับฉลากทำนอกระบบ** (โทร, Line, พบหน้า)

```
ท้าวแชร์จัดประมูล → ได้ผู้ชนะ → Admin บันทึกลงระบบ → ระบบคำนวณเงิน
```

**กฎประมูล:** คนใส่ดอกน้อยสุด = ชนะ

---

## การคำนวณเงิน
### เงินที่ได้รับ
```
เงินที่ได้ = เงินกองกลาง - ดอกเบี้ย - รายการหักรับ
```

---

## รายการหักรับ

เงินที่หักจากผู้รับเงินก้อน:

| รายการ | คำอธิบาย |
|--------|----------|
| **ดอกเบี้ย** | เงินที่เสนอประมูล |
| **ค่าดูแลวง** | ค่าบริหารจัดการ (ถ้ามี) |
| **หักท้ายท้าว** | ค่าตอบแทนท้าวแชร์ |
| **ค่าดำเนินการ** | ค่าใช้จ่ายอื่นๆ |

---

## ตัวอย่าง (5 คน, ส่ง 2,000 บาท)

| งวด | ผู้ชนะ | ดอกเบี้ย | ได้รับ |
|-----|--------|----------|--------|
| 1 | ท้าวแชร์ | - | 10,000 |
| 2 | สมาชิก A | 50 | 9,950 |
| 3 | สมาชิก B | 60 | 9,940 |
| 4 | สมาชิก C | 40 | 9,960 |
| 5 | สมาชิก D | - | 10,000 |

---

## บทบาท

### ท้าวแชร์
- สร้างวง กำหนดกติกา
- เก็บเงิน ทวงเงิน
- **ได้เงินงวดแรก** (สิทธิพิเศษ)

### ลูกแชร์
- ส่งเงินทุกงวด
- ประมูลเมื่อต้องการเงิน

---

## ข้อมูลสมาชิก

| ฟิลด์ | บังคับ | คำอธิบาย |
|-------|--------|----------|
| **รหัสลูกแชร์** | Auto | สร้างอัตโนมัติ: A, B, C, ... Z, AA, AB, ... |
| **ชื่อเล่น** | ใช่ | ชื่อที่ใช้เรียกในวง |
| ที่อยู่ | ไม่ | ที่อยู่ลูกแชร์ |
| เบอร์โทร | ไม่ | เบอร์ติดต่อ |
| ไลน์ไอดี | ไม่ | LINE ID |
