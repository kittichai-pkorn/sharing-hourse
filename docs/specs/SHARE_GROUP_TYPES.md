| กฎ | รายละเอียด |
|----|------------|
| งวดแรก | **ท้าวแชร์ได้เงินเสมอ** (ไม่ประมูล/จับฉลาก) |

## รายการหักรับ (ทุกประเภท)

เงินที่หักจากผู้รับเงินก้อน:

| รายการ | คำอธิบาย |
|--------|----------|
| **ดอกเบี้ย** | เงินที่เสนอประมูล หรือดอกเบี้ยคงที่ |
| **ค่าดูแลวง** | ค่าบริหารจัดการ (ถ้ามี) |
| **หักท้ายท้าว** | ค่าตอบแทนท้าวแชร์ |
| **ค่าดำเนินการ** | ค่าใช้จ่ายอื่นๆ |

### สูตรคำนวณ
```
เงินที่ได้รับ = เงินกองกลาง - ดอกเบี้ย - ค่าดูแลวง - หักท้ายท้าว - ค่าดำเนินการ
```

## 1. ขั้นบันได

### คำอธิบาย
ดอกเบี้ยคงที่ทุกงวด ใช้วิธี **จับฉลาก** เลือกผู้ชนะ

### วิธีเลือกผู้ชนะ
```
งวดแรก → ท้าวแชร์ได้เงิน
งวดถัดไป → จับฉลาก (นอกระบบ) → Admin บันทึกผล
```

### ฟิลด์เฉพาะ
| ฟิลด์ | คำอธิบาย |
|-------|----------|
| ดอกเบี้ยคงที่ | ดอกเบี้ยที่หักทุกงวด |

### ตัวอย่าง (5 คน, เงินต้น 2,000 บาท, ดอกเบี้ย 100 บาท)

| งวด | ผู้ชนะ | ดอกเบี้ย | ได้รับ |
|-----|--------|----------|--------|
| 1 | ท้าวแชร์ | 100 | 9,900 |
| 2 | จับฉลาก | 100 | 9,900 |
| 3 | จับฉลาก | 100 | 9,900 |
| 4 | จับฉลาก | 100 | 9,900 |
| 5 | คนสุดท้าย | - | 10,000 |

### รายการหักรับ
```
เงินกองกลาง:    10,000 บาท
- ดอกเบี้ย:       -100 บาท (คงที่)
- ค่าดูแลวง:      -50 บาท (ถ้ามี)
- หักท้ายท้าว:   -200 บาท (ถ้ามี)
───────────────────────
ได้รับจริง:      9,650 บาท
```

## 2. บิทดอกตาม

### คำอธิบาย
**ประมูลดอกเบี้ย** - คนใส่ดอกน้อยสุดชนะ

### วิธีเลือกผู้ชนะ
```
งวดแรก → ท้าวแชร์ได้เงิน
งวดถัดไป → ประมูล (นอกระบบ) → Admin บันทึกผู้ชนะ + ดอกเบี้ย
```

### ฟิลด์เฉพาะ
| ฟิลด์ | คำอธิบาย |
|-------|----------|
| ส่งต่องวด | จำนวนงวดที่ต้องส่งก่อนเริ่มประมูล |

### ตัวอย่าง (5 คน, เงินต้น 2,000 บาท)

| งวด | ผู้ชนะ | ประมูลดอก | ได้รับ |
|-----|--------|-----------|--------|
| 1 | ท้าวแชร์ | - | 10,000 |
| 2 | คน A (ดอกน้อยสุด) | 50 | 9,950 |
| 3 | คน B (ดอกน้อยสุด) | 60 | 9,940 |
| 4 | คน C (ดอกน้อยสุด) | 40 | 9,960 |
| 5 | คนสุดท้าย | - | 10,000 |

### รายการหักรับ
```
เงินกองกลาง:    10,000 บาท
- ดอกเบี้ย:       -50 บาท (ตามประมูล)
- ค่าดูแลวง:      -50 บาท (ถ้ามี)
- หักท้ายท้าว:   -200 บาท (ถ้ามี)
───────────────────────
ได้รับจริง:      9,700 บาท
```

## 4. บิทลดต้น - ลดต้นงวดถัดไป

### คำอธิบาย
**ประมูลดอกเบี้ย** - ดอกที่ประมูลจะลดเงินต้นของ **งวดถัดไป**

### วิธีเลือกผู้ชนะ
```
งวดแรก → ท้าวแชร์ได้เงิน
งวดถัดไป → ประมูล (นอกระบบ) → Admin บันทึกผู้ชนะ + ดอกเบี้ย
```

### ฟิลด์เฉพาะ
| ฟิลด์ | คำอธิบาย |
|-------|----------|
| ส่งต่องวด | จำนวนงวดที่ต้องส่งก่อน |

### ตัวอย่าง (5 คน, เงินต้น 2,000 บาท)
```
งวดที่ 2: ประมูลชนะ 100 บาท
- ได้รับ = 10,000 - 100 = 9,900 บาท
- ลดเงินต้นงวดถัดไป = 100 ÷ 4 = 25 บาท/คน

งวดที่ 3: เงินต้นใหม่ = 2,000 - 25 = 1,975 บาท
```

### รายการหักรับ
```
เงินกองกลาง:    10,000 บาท
- ดอกเบี้ย:       -100 บาท (ตามประมูล)
- ค่าดูแลวง:      -50 บาท (ถ้ามี)
- หักท้ายท้าว:   -200 บาท (ถ้ามี)
───────────────────────
ได้รับจริง:      9,650 บาท
```

### ข้อดี
เงินที่ต้องส่งลดลงเรื่อยๆ

## 5. บิทลดต้น - ลดต้นงวดที่บิท

### คำอธิบาย
**ประมูลดอกเบี้ย** - ดอกที่ประมูลจะลดเงินต้นของ **งวดนั้นทันที**

### วิธีเลือกผู้ชนะ
```
งวดแรก → ท้าวแชร์ได้เงิน
งวดถัดไป → ประมูล (นอกระบบ) → Admin บันทึกผู้ชนะ + ดอกเบี้ย
```

### ฟิลด์เฉพาะ
| ฟิลด์ | คำอธิบาย |
|-------|----------|
| ส่งต่องวด | จำนวนงวดที่ต้องส่งก่อน |

### ตัวอย่าง (5 คน, เงินต้น 2,000 บาท)
```
งวดที่ 2: ประมูลชนะ 100 บาท
- ลดเงินต้นทันที = 100 ÷ 4 = 25 บาท/คน
- สมาชิกจ่าย = 2,000 - 25 = 1,975 บาท
- ได้รับ = 1,975 × 4 + 2,000 = 9,900 บาท
```

### รายการหักรับ
```
เงินกองกลาง:    9,900 บาท (หลังลดต้น)
- ค่าดูแลวง:      -50 บาท (ถ้ามี)
- หักท้ายท้าว:   -200 บาท (ถ้ามี)
───────────────────────
ได้รับจริง:      9,650 บาท
```

### ความแตกต่าง
สมาชิกได้ประโยชน์ทันทีในงวดที่ประมูล
