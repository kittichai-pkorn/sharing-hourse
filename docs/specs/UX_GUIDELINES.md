# UX Guidelines - ระบบวงแชร์

---

## 1. Dashboard (หน้าหลัก)

### แสดงข้อมูลสำคัญ
```
┌─────────────────────────────────────────────────────┐
│  📊 ภาพรวม                                          │
├─────────────────────────────────────────────────────┤
│  วงแชร์ทั้งหมด: 5        งวดที่ต้องดำเนินการ: 3     │
│  กำลังดำเนินการ: 3       รอเก็บเงิน: 2              │
└─────────────────────────────────────────────────────┘
```

### การ์ดวงแชร์
```
┌─────────────────────────────────────┐
│  วงแชร์พนักงาน มกราคม 2568          │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│  ประเภท: บิทดอกตาม                  │
│  สมาชิก: 10/10 คน                   │
│  งวดปัจจุบัน: 3/10                  │
│  ▓▓▓▓▓▓▓▓▓░░░░░░░░░░░░░░░░░ 30%    │
│                                     │
│  สถานะ: 🟢 กำลังดำเนินการ           │
│                      [ดูรายละเอียด] │
└─────────────────────────────────────┘
```

### คำแนะนำ
- ใช้สีแยกสถานะ: 🟢 ดำเนินการ, 🟡 รอเปิด, 🔴 มีปัญหา
- แสดง Progress bar ความคืบหน้าวง
- Quick action: ปุ่มลัดไปงวดที่ต้องดำเนินการ

## 2. สร้างวงแชร์

### Wizard แบบ Step-by-step
```
Step 1          Step 2          Step 3          Step 4
[ข้อมูลพื้นฐาน] → [ตั้งค่าวง] → [รายการหักรับ] → [ตรวจสอบ]
    ●──────────────○──────────────○──────────────○
```

### Step 1: ข้อมูลพื้นฐาน
```
┌─────────────────────────────────────────────────────┐
│  ชื่อวงแชร์ *                                       │
│  ┌─────────────────────────────────────────────┐   │
│  │ วงแชร์พนักงาน มกราคม 2568                   │   │
│  └─────────────────────────────────────────────┘   │
│                                                     │
│  รูปแบบวงแชร์ *                                    │
│  ┌─────────────────────────────────────────────┐   │
│  │ บิทดอกตาม                              ▼   │   │
│  └─────────────────────────────────────────────┘   │
│  ℹ️ คนใส่ดอกน้อยสุดชนะ                             │
│                                                     │
│                              [ยกเลิก] [ถัดไป →]    │
└─────────────────────────────────────────────────────┘
```

### Step 2: ตั้งค่าวง
```
┌─────────────────────────────────────────────────────┐
│  จำนวนสมาชิก *              เงินต้นต่องวด *         │
│  ┌───────────────┐          ┌───────────────┐      │
│  │ 10        คน │          │ 2,000     บาท │      │
│  └───────────────┘          └───────────────┘      │
│                                                     │
│  📋 สรุป: เงินกองกลาง = 2,000 × 10 = 20,000 บาท   │
│                                                     │
│  รอบการชำระ *                                      │
│  ○ รายวัน  ● รายเดือน  ○ ราย ___ วัน              │
│                                                     │
│  วันที่เริ่มต้น *                                   │
│  ┌───────────────┐                                 │
│  │ 01/01/2568 📅 │                                 │
│  └───────────────┘                                 │
│                                                     │
│                         [← ก่อนหน้า] [ถัดไป →]     │
└─────────────────────────────────────────────────────┘
```

### Step 3: รายการหักรับ
```
┌─────────────────────────────────────────────────────┐
│  รายการหักรับ (ไม่บังคับ)              [+ เพิ่ม]   │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│                                                     │
│  │ # │ รายการ          │ จำนวน      │        │    │
│  ├───┼─────────────────┼────────────┼────────┤    │
│  │ 1 │ ค่าดูแลวง       │ 100 บาท    │ [🗑️]  │    │
│  │ 2 │ หักท้ายท้าว     │ 200 บาท    │ [🗑️]  │    │
│  └───┴─────────────────┴────────────┴────────┘    │
│                                                     │
│  ยอดรวมหักรับ: 300 บาท/งวด                         │
│                                                     │
│                         [← ก่อนหน้า] [ถัดไป →]     │
└─────────────────────────────────────────────────────┘
```

### Step 4: ตรวจสอบก่อนบันทึก
```
┌─────────────────────────────────────────────────────┐
│  ✅ ตรวจสอบข้อมูล                                   │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│                                                     │
│  ชื่อวง:        วงแชร์พนักงาน มกราคม 2568          │
│  รูปแบบ:        บิทดอกตาม                          │
│  สมาชิก:        10 คน                              │
│  เงินต้น:       2,000 บาท                          │
│  เงินกองกลาง:   20,000 บาท                         │
│  รอบชำระ:       รายเดือน                           │
│  เริ่มต้น:      01/01/2568                         │
│  รายการหักรับ:  300 บาท                            │
│                                                     │
│  ⚠️ หลังบันทึกแล้วจะแก้ไขบางข้อมูลไม่ได้           │
│                                                     │
│                      [← ก่อนหน้า] [💾 บันทึก]      │
└─────────────────────────────────────────────────────┘
```

### คำแนะนำ
- แสดง Tooltip อธิบายแต่ละประเภทวง
- Real-time calculation แสดงเงินกองกลาง
- Validation ทันทีเมื่อกรอกข้อมูล
- ปุ่มย้อนกลับได้ทุก step

## 3. จัดการสมาชิก

### รายการสมาชิก
```
┌─────────────────────────────────────────────────────┐
│  สมาชิก (8/10)                         [+ เพิ่ม]   │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│                                                     │
│  │ # │ ชื่อ          │ บทบาท    │ สถานะ   │      │ │
│  ├───┼───────────────┼──────────┼─────────┼──────┤ │
│  │ 1 │ 👑 สมชาย     │ ท้าวแชร์  │ เปียแล้ว │ ⋮   │ │
│  │ 2 │ สมหญิง       │ ลูกแชร์   │ ยังไม่เปีย│ ⋮   │ │
│  │ 3 │ สมศักดิ์      │ ลูกแชร์   │ เปียแล้ว │ ⋮   │ │
│  └───┴───────────────┴──────────┴─────────┴──────┘ │
│                                                     │
│  🟢 เปียแล้ว: 2    🔵 ยังไม่เปีย: 6               │
└─────────────────────────────────────────────────────┘
```

### เพิ่มสมาชิก
```
┌─────────────────────────────────────────────────────┐
│  เพิ่มสมาชิกใหม่                              [✕]  │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│                                                     │
│  🔍 ค้นหาสมาชิก                                    │
│  ┌─────────────────────────────────────────────┐   │
│  │ พิมพ์ชื่อหรือเบอร์โทร...                    │   │
│  └─────────────────────────────────────────────┘   │
│                                                     │
│  หรือ                                              │
│                                                     │
│  [+ สร้างสมาชิกใหม่]                               │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### คำแนะนำ
- แสดง icon 👑 สำหรับท้าวแชร์
- Filter: ทั้งหมด / เปียแล้ว / ยังไม่เปีย
- Quick search ค้นหาสมาชิก
- Drag & drop เรียงลำดับ (ถ้าจำเป็น)

│  งวดทั้งหมด                                         │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │

│  │ สมศรี                                   ▼   │   │
│  └─────────────────────────────────────────────┘   │

│                                                     │
│  ┌─────────────────────────────────────────────┐   │
│  │  📋 สรุป                                    │   │
│  │  เงินกองกลาง:    20,000 บาท                 │   │

- Dropdown แสดงเฉพาะคนที่ยังไม่เปีย
- Real-time คำนวณเงินที่ได้รับ
- แสดงรายละเอียดรายการหักรับทั้งหมด
- Confirmation ก่อนบันทึก

## 5. รายการหักรับ

### แสดงรายการหักรับของงวด
```
┌─────────────────────────────────────────────────────┐
│  รายการหักรับ - งวดที่ 3                           │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│                                                     │
│  │ รายการ              │ จำนวน      │ สถานะ   │   │
│  ├─────────────────────┼────────────┼─────────┤   │
│  │ ดอกเบี้ย            │ 80 บาท     │ ✅      │   │
│  │ ค่าดูแลวง           │ 100 บาท    │ ✅      │   │
│  │ หักท้ายท้าว         │ 200 บาท    │ ✅      │   │

- แสดงยอดรวมและเงินที่ได้รับจริง
- สามารถแก้ไขได้ (ถ้ายังไม่ปิดงวด)

## 6. Mobile UX

### หน้าจอมือถือ
```
┌─────────────────────┐
│ ☰  วงแชร์     🔔   │
├─────────────────────┤
│                     │
│ 📊 ภาพรวม          │
│ ┌─────────────────┐ │
│ │ วงทั้งหมด: 5    │ │
│ │ รอดำเนินการ: 3  │ │
│ └─────────────────┘ │
│                     │
│ วงแชร์ของฉัน        │
│ ┌─────────────────┐ │
│ │ 🟢 วงพนักงาน   │ │
│ │    งวด 3/10    │ │
│ │    ▓▓▓░░░ 30%  │ │
│ └─────────────────┘ │
│                     │
│ ┌─────────────────┐ │
│ │ 🟢 วงครอบครัว  │ │
│ │    งวด 5/8     │ │
│ │    ▓▓▓▓▓░ 62%  │ │
│ └─────────────────┘ │
│                     │
├─────────────────────┤
│ 🏠    📋    ➕    👤 │
└─────────────────────┘
```

### Bottom Navigation
```
🏠 หน้าหลัก | 📋 วงแชร์ | ➕ สร้างวง | 👤 โปรไฟล์
```

### คำแนะนำ
- ใช้ Bottom navigation สำหรับ mobile
- Card layout สำหรับแสดงวงแชร์
- Swipe actions: ปัดซ้าย = ลบ, ปัดขวา = แก้ไข
- Pull to refresh
- Large touch targets (min 44px)

## 7. Notifications

### ประเภท Notifications
| เหตุการณ์ | ข้อความ |
|-----------|---------|
| งวดใหม่ | 📅 งวดที่ 3 ของ "วงพนักงาน" เริ่มแล้ว |
| รอบันทึกผู้ชนะ | ⏰ รอบันทึกผู้ชนะงวดที่ 3 |
| บันทึกสำเร็จ | ✅ บันทึกผู้ชนะงวดที่ 3 เรียบร้อย |
| วงปิด | 🎉 วง "วงพนักงาน" ครบทุกงวดแล้ว |

### คำแนะนำ
- Push notification สำหรับเหตุการณ์สำคัญ
- In-app notification center
- อ่าน/ยังไม่อ่าน status

## 8. Error Handling

### แสดง Error
```
┌─────────────────────────────────────────────────────┐
│  ⚠️ ไม่สามารถบันทึกได้                              │
│                                                     │
│  สมาชิกคนนี้เปียไปแล้วในงวดที่ 2                    │
│  กรุณาเลือกสมาชิกคนอื่น                             │
│                                                     │
│                                        [ตกลง]      │
└─────────────────────────────────────────────────────┘
```

### Validation
```
┌─────────────────────────────────────────────────────┐
│  จำนวนสมาชิก *                                     │
│  ┌─────────────────────────────────────────────┐   │
│  │ 2                                           │   │
│  └─────────────────────────────────────────────┘   │
│  ❌ จำนวนสมาชิกต้องมีอย่างน้อย 3 คน                │
└─────────────────────────────────────────────────────┘
```

### คำแนะนำ
- แสดง error message ชัดเจน เข้าใจง่าย
- Inline validation ทันที
- แนะนำวิธีแก้ไข

```
┌─────────────────────────────────────────────────────┐

│                     📋                              │
│                                                     │
│              ยังไม่มีวงแชร์                         │
│         เริ่มสร้างวงแชร์แรกของคุณ                   │

│              [+ สร้างวงแชร์ใหม่]                    │
│                                                     │

### คำแนะนำ
- แสดง illustration/icon
- ข้อความอธิบายชัดเจน
- CTA ชัดเจน
