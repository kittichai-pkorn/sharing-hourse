## Info
| Field | Value |
|-------|-------|

| Priority | P1 |
| Points | 2 (S) |

---

## User Story

```
As a: ระบบ
I want to: กำหนดคนสุดท้ายเป็นผู้ชนะอัตโนมัติ
So that: ตามกฎคนสุดท้ายได้เงินเต็ม
```

---

## Acceptance Criteria

- [ ] เหลือคนเดียว = ผู้ชนะอัตโนมัติ (auto-select)
- [ ] ไม่มีดอกเบี้ย (= 0) / disabled field
- [ ] ได้เงินเต็ม (หักเฉพาะค่าดูแล/ค่าดำเนินการ)
- [ ] Admin กดยืนยันเท่านั้น
- [ ] แสดงข้อความ "งวดสุดท้าย - ได้รับเงินเต็ม"

## Logic

- ตรวจสอบ คนที่ยังไม่เปีย === 1
- Auto-select คนที่เหลือ
- ดอกเบี้ย = 0 (fixed)
- หลังยืนยัน → ปิดวง (สถานะ = COMPLETED)
