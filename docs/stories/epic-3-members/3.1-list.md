
## Acceptance Criteria

- [x] แสดงรายการลูกแชร์ (รหัส, ชื่อเล่น, เบอร์โทร, ไลน์)
- [x] รหัสลูกแชร์ (memberCode) แสดงเป็น A001, A002, A003, ...
