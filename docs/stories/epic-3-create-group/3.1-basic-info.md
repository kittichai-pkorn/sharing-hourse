## ประเภทวง (Dropdown options)

| ค่า | ชื่อ | คำอธิบาย |
|-----|------|----------|
| STEP_INTEREST | ขั้นบันได | ดอกเบี้ยคงที่ + จับฉลาก |
| BID_INTEREST | บิทดอกตาม | ประมูลดอก คนใส่น้อยสุดชนะ |
| FIXED_INTEREST | ดอกตาม | ดอกเบี้ยคงที่ + จับฉลาก |
