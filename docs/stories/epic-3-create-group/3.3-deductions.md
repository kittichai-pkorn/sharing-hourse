## User Story

```
As a: ท้าวแชร์
I want to: กำหนดรายการหักรับ
So that: หักค่าใช้จ่ายจากผู้ชนะ
```

## Acceptance Criteria

- [ ] เพิ่มรายการหักรับ (ชื่อ + จำนวน)
- [ ] ลบรายการหักรับ
- [ ] แก้ไขรายการหักรับ
- [ ] แสดงยอดรวมหักรับ
- [ ] รายการ default: ค่าดูแลวง (จาก step 2)
