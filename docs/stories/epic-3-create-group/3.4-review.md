| Field | Value |

| Priority | P0 |

---

## User Story

```
As a: ท้าวแชร์
I want to: ตรวจสอบข้อมูลก่อนบันทึก
So that: มั่นใจว่าข้อมูลถูกต้อง
```

---

## Acceptance Criteria

- [ ] แสดงสรุปข้อมูลทั้งหมด
- [ ] แสดงคำเตือน "แก้ไขบางข้อมูลไม่ได้หลังบันทึก"
- [ ] ปุ่มย้อนกลับแก้ไข
- [ ] ปุ่มบันทึก
- [ ] สร้างวงแชร์
- [ ] ท้าวแชร์ = คนสร้างวง
- [ ] Redirect ไปหน้าวงที่สร้าง

## หลังบันทึก

- สร้าง ShareGroup
- เพิ่ม Host (ท้าวแชร์) = User ที่สร้าง
- สร้าง Deductions
- สถานะ = PENDING (รอเปิด)
