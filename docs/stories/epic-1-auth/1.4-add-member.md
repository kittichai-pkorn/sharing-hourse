# Story 1.4: เพิ่มสมาชิก (ลูกแชร์) เข้า Tenant

## Info
| Field | Value |
|-------|-------|
| Epic | 1. Authentication |
| Priority | P1 |
| Points | 3 (M) |

---

## User Story

```
As a: ท้าวแชร์ (Admin)
I want to: เพิ่มลูกแชร์เข้า Tenant
So that: ลูกแชร์สามารถเข้าระบบได้
```

---

## Acceptance Criteria

- [ ] Admin เท่านั้นที่เพิ่มได้
- [ ] กรอกข้อมูลสมาชิกใหม่
- [ ] สร้าง User ใน Tenant เดียวกัน
- [ ] Role = USER (ลูกแชร์)
- [ ] ส่ง SMS/Email แจ้งรหัสผ่านชั่วคราว (optional)

## API

```typescript
POST /api/tenants/:tenantId/users

// Request
{
  fullName: string;
  phone: string;
  email?: string;
  password: string;
}

// Response
{
  success: true,
  data: {
    id: string;
    fullName: string;
    phone: string;
    email: string;
    role: "USER";
  }
}
```

## Notes

- ลูกแชร์ login ด้วย: รหัสวง + เบอร์โทร + รหัสผ่าน
- Admin สามารถ reset password ให้ลูกแชร์ได้
