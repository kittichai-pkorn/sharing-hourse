## Current Status

ปัจจุบัน **ลูกแชร์ไม่สามารถ login ได้** เพราะ:
- ท้าวแชร์จัดการวงให้ทั้งหมด
- ไม่จำเป็นต้องมี User account สำหรับลูกแชร์

**ถ้าต้องการในอนาคต:**
- เปิด login สำหรับ USER role
- ให้ลูกแชร์ดูข้อมูลวงของตัวเอง

## Implementation (ถ้าต้องการ)

### API
```typescript
POST /api/users

// Request
{
  firstName: string;
  lastName: string;
  phone: string;
  email?: string;
  password: string;
}

// Response
{
  success: true,
  data: {
    id: number;
    firstName: string;
    lastName: string;
    phone: string;
    email: string | null;
    role: "USER";
  }
}
```

### Files (มีอยู่แล้ว แต่ไม่ได้ใช้)
- `backend/src/routes/users.ts` - CRUD Users
- `frontend/src/pages/dashboard/UsersPage.tsx` - UI จัดการ Users
