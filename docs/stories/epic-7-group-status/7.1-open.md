## User Story

```
As a: ท้าวแชร์
I want to: เปิดวงเมื่อสมาชิกครบ
So that: เริ่มดำเนินการวงได้
```

---

## Acceptance Criteria

- [ ] ตรวจสอบสมาชิกครบตามจำนวน
- [ ] เปลี่ยนสถานะเป็น "กำลังดำเนินการ"
- [ ] สร้างงวดทั้งหมดอัตโนมัติ
- [ ] งวดแรก → ผู้ชนะ = ท้าวแชร์
- [ ] คำนวณวันกำหนดชำระแต่ละงวด
- [ ] Confirmation ก่อนเปิด
