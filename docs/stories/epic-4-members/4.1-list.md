## User Story

```
As a: ท้าวแชร์
I want to: เห็นรายการสมาชิกในวง
So that: รู้ว่าใครอยู่ในวงและสถานะเปีย
```

---

## Acceptance Criteria

- [x] แสดงรายการสมาชิก (รหัส, ชื่อเล่น, สถานะ, เบอร์โทร)
- [x] แสดงสถานะ เปียแล้ว/ยังไม่เปีย
- [x] แสดงงวดที่เปีย (ถ้าเปียแล้ว)
- [x] แสดงจำนวน สมาชิก/สูงสุด
- [x] แสดงสรุป เปียแล้ว vs ยังไม่เปีย
