## Acceptance Criteria

- [ ] แสดงรายการสมาชิก (ลำดับ, ชื่อ, บทบาท, สถานะ)
- [ ] แสดง icon 👑 ท้าวแชร์
- [ ] แสดงสถานะ เปีย/ยังไม่เปีย
- [ ] แสดงจำนวน สมาชิก/สูงสุด
- [ ] แสดงจำนวน เปียแล้ว vs ยังไม่เปีย
