## Error Cases

| กรณี | ข้อความ |
|------|---------|
| อยู่ในวงแชร์ | ไม่สามารถลบลูกแชร์ที่อยู่ในวงแชร์ได้ |

---

## API

```typescript
DELETE /api/members/:id

// Response
{
  success: true,
  message: "ลบลูกแชร์เรียบร้อยแล้ว"
}

// Error Response (อยู่ในวง)
{
  success: false,
  error: "ไม่สามารถลบลูกแชร์ที่อยู่ในวงแชร์ได้"
}
```
