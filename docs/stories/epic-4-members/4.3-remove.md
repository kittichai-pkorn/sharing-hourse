## User Story

```
As a: ท้าวแชร์
```

## Acceptance Criteria

- [ ] ลบสมาชิกที่ยังไม่เปีย
- [ ] ไม่ให้ลบคนที่เปียแล้ว
- [ ] ไม่ให้ลบท้าวแชร์
- [ ] ไม่ให้ลบถ้าวงเปิดแล้ว
- [ ] Confirmation ก่อนลบ

## Error Cases

| กรณี | ข้อความ |
|------|---------|
| เปียแล้ว | ไม่สามารถลบสมาชิกที่เปียแล้วได้ |
| ท้าวแชร์ | ไม่สามารถลบท้าวแชร์ได้ |
| วงเปิดแล้ว | ไม่สามารถลบสมาชิกหลังวงเปิดแล้ว |
